Программа курса «Fashion Stylist»

Общая продолжительность курса: 42 академических часов

Блок 1. Работа стилиста
7 занятий по 1,5 часа

    Занятие 1. Суть работы стилиста

    Занятие 2. Рынок моды и его основные игроки.

    Занятие 3. Как появляются модные тренды. Модные тенденции текущего сезона.

    Занятие 4. История моды от античности до сегодняшних дней.

    Занятие 5. Модные столицы мира и их особенности. Недели моды.

    Занятие 6. Тренд-бук стилиста: его разработка и использование. Как всегда быть в тренде.

    Занятие 7. Как смешивать масс маркет и люкс.

Блок 2. Типы внешности. Колористика
6 занятий по 1,5 часа

    Занятие 1. Различные типы внешности. Отличия.

    Занятие 2. Стилистические направления

    Занятие 3. Колористика. Психология цвета. Цвет и имидж.

    Занятие 4. Искусство цвета и лучшие цветовые сочетания.

    Занятие 5. Цвет в гардеробе клиента. Индивидуальные цветовые колориты.

    Занятие 6. Составление индивидуальных палитр.

Блок 3. Шопинг в ТЦ Санкт-Петербурга.
6 занятий по 1,5 часа

Часть 1 (теория):

    Занятие 1. Типы фигур. Примеры одежды для каждого типа. Анализ фигуры и композиция образа.

    Занятие 2. Визуальная коррекция фигуры с помощью одежды. Практика подбора одежды для разного типа фигур. Многослойные образы: как их составлять.

    Занятие 3. Коррекция черт лица, с помощью макияжа.

    Часть 2 (практика):

    Занятие 1. Подготовка программы работы с клиентом. Разбор гардероба клиента.

    Занятие 2. Отработка практических навыков — шоппинг совместно с моделью.

    Занятие 3. Отработка практических навыков — шоппинг совместно с моделью.

Блок 4. Практика: шоппинг в торговом комплексе
7 занятий по 1,5 часа

    Занятие 1. Подготовка к шопингу — установка целей и задач.

    Занятие 2. Особенности онлайн и офлайн шопинга.

    Занятие 3. Учимся читать этикетки: как определять качество вещей во время покупок.

    Занятие 4. Практика подбора одежды для разного типа фигур.

    Занятие 5. Составление стилевых комплектов для реальных людей.

    Занятие 6. Разработка стилевого направления для вас лично.

    Занятие 7. Bonus: список магазинов для различных стилей.

Блок 5. Маркетинг для стилиста. Создание и построение персонального бренда
4 занятия - 1,5 часа

    Занятие 1. Основы брендинга.

    Занятие 2. Технология построения личного бренда. Создание УТП

    Занятие 2. Представление услуг в виде товара

    Занятие 2. Коммуникации с будущими клиентами.
